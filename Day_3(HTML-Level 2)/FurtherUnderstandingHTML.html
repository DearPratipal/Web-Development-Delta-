Understanding HTML 5

HTML 5 is a markup language used for structuring and presenting content on the web. It is the fifth major version of the HTML standard. HTML 5 is designed to be backwards compatible with older browsers, while also providing new features and functionality that take advantage of modern web technologies.

How HTML works?
